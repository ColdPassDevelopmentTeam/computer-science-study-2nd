# Computer Science Study 2nd

알고 있는 Java 백엔드 지식을 결론부터 자기 언어로 설명하고, 조건·대안·실패와 운영 책임까지
답변을 확장하는 스터디입니다. 연차나 사람의 등급을 평가하지 않고 기술적 주장과 판단 과정을
훈련합니다.

## 공통 목표

- 매주 작은 범위를 공부하고 내부 동작을 자기 언어로 설명하기
- 답이 성립하는 조건과 모르는 부분을 구분하고 대안의 trade-off 비교하기
- 기술 면접의 꼬리질문을 선택 근거와 운영 책임까지 확장하기

## 운영안 선택

한 과정에서는 운영안 하나만 선택하며 같은 세션에서 두 안을 섞지 않습니다.

| 구분 | 1안 | 2안 |
|---|---|---|
| 핵심 목표 | 학습 습관과 설명 훈련 | 실전 면접과 깊이 검증 |
| 기간 | 16주 | 4주 |
| 진행 | 같은 질문을 공부하고 주간 TIL로 랜덤 발표 | 비공개 질문을 Depth 5~7까지 이어가고 핵심 오답 보충 |
| 시작 기준 | 기본 개념과 설명 습관이 필요한 경우 | 공개 키워드를 자료 없이 2~3분 설명할 수 있는 경우 |
| 문서 | [운영 규칙](./plan-1/) · [커리큘럼](./plan-1/curriculum/) | [운영 규칙](./plan-2/) · [커리큘럼](./plan-2/curriculum/) |

## 공통 운영

- **일정 기준:** 주간 범위와 일정은 Discord 공지 채널을 기준으로 합니다.
- **평일 모각코:** 평일 20시에 가능한 사람이 여는 자유 참여 공간이며 출석으로 평가하지 않습니다.
- **주말 세션:** 시간과 종료 기준은 선택한 운영안의 `README.md`를 따릅니다.
- **AI 사용:** 학습과 문서 정리에 사용할 수 있지만 작성한 내용은 직접 설명해야 합니다.
- **결석·취소:** 별도 과제나 밀린 산출물로 대체하지 않습니다.
- **세부 규칙:** 준비·발표·면접·오답 보충은 선택한 운영안의 `README.md`를 따릅니다.
